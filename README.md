# Clasificación de Imágenes de Cartas usando Deep Learning

## Descripción del Proyecto
Una implementación progresiva de deep learning para la clasificación de cartas de juego utilizando:
- Arquitectura CNN Básica (v1)
- Transfer Learning con MobileNetV2 (v2)
- Agregar 2 capas densas (v3)
- Aplicar más variabilidad con Data Augmentation (v4)
- Modificación del Dropout (v5)
- Aplicación de Learning Rate adaptativo y Early Stopping (v6)
